Graph

trait Graph extends AnyRef

Generic directed graph base-type

Provides basic graph functionalities such as - nodes, the set of nodes in this graph - edges, the set of edges in this graph - firstNode and lastNode, the respective first and last nodes of this graph (we require these to exist in our use case) - next, prev, transitiveNext, transitivePrev and transitiveEdge graph connectedness functions - map that transforms the nodes in the graph - some pretty-printing utilities such as printStructure
