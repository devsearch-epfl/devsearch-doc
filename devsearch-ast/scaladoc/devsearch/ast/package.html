package ast

The DevSearch common AST format.

The devsearch projects provides a common AST format that unifies languages into a single abstract representation. The base AST type is ast.AST and additional information about input sources, positions, comments and operations on ASTs can be found at:

  • ast.Source
  • ast.Positional
  • ast.Commentable
  • ast.Operators
