package parsers

Language parsers supported by the DevSearch project.

Parsers must conform to the parsers.Parser interface that requires the methods

def language: String
def parse(source: Source): AST

to be defined. In case of parsing failures, the parse method should throw parsers.ParsingFailedError to provide information about invalid source code.

To enable language-agnostic parsing, the parsers.Languages.orderedLangList value should be extended with any new parser additions.
